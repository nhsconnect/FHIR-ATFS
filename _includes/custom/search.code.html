{{include.para}} code

GET [baseUrl]/{{include.content}}?patient=[id]&code=[system]|[code]
GET [baseUrl]/{{include.content}}?patient=[id]&code=[code]

See token for details on this parameter. ‘code’ can be used multiple times as a search parameter

system is a uniform resource identifier which defines which CodeSystem the code belongs to. This is optional and if not present all matching codes will be returned regardles of CodeSystems.

To search for {{include.content}}s with Baseline body mass index using SNOMED CT

GET [baseUrl]/{{include.content}}?patient=[id]&code=http://snomed.info/sct|846931000000101

Return all {{include.content}} resources with SNOMED CT code of 846931000000101 and Patient with a logical id of 1.

GET [baseUrl]/{{include.content}}?patient=1&code=846931000000101

Return all {{include.content}} resources with code of 846931000000101 from any CodeSystem and Patient with a logical id of 1.
